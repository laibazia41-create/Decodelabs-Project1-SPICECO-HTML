# SPICECO - Semantic HTML5 Architecture (Project 1)

Developed during the Industrial Training Engagement Phase with **DecodeLabs** (Batch: 2026).

## 🚀 Project Overview
This repository contains the foundational milestone for the **SPICECO** premium storefront. The core objective of this project was to establish a solid structural blueprint without relying on styling hacks or complex layouts, moving away from loose design elements toward strict structural compliance.

## 🛠️ Tech Stack Used
- **HTML5:** 100% Native semantic structure.

## 📐 Key Specifications Implemented
- **Semantic Rooting:** Leveraged structural tags (`<header>`, `<main>`, `<section>`, `<article>`, and `<footer>`) to construct an accessible DOM outline.
- **Asset Mapping:** Embedded raw visual components using proper container wrapping and descriptive `alt` text parameters for SEO visibility.
- **Resource Reference Paths:** Organized structural paths link points for internal styles and document navigation cleanly.
